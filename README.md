1. Includes Creation of New Userof type EXTERNAL
2. They Can Login with Gmail and password (Using OTP)
3. IITD Login system included to Cater IITD login
4. Clients can be registered for auth system (BSW Website)
5. Clients will have clientID, ClientSecret, redirectURLs and Server URI
6. Clients can be managed from frontend
